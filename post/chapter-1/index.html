Project 1: Predicting Energy Shortfall

May 25, 2023

As the world grapples with the crisis of climate change many countries are looking into moving more towards clean energy, I was working in a team that was given a project to work on predicting a load shortfall between electricity produced by fossil fuels and renewables.

  • We were given a dataset with historical weather conditions in different cities in Spain as features and the Load Shortfall.

  • We began with cleaning our dataset to ensure that our analysis and model will be accurate.

  • Worked on visualisations to see our dataset which helped us picking up patterns and outliers in the data, worked on figuring out how to deal with such.

  • Correlation was important for us in identifying which features will be included when building our model, we produced a correlation matrix which would help us with multicollinearity, identifying predictor variable that are highly correlated and dealing with the, it assisted us with feature selection we took features that are highly correlated with the target variable to produce our model.

  • Having had all the required features that will be useful to buidling our model we had to start developing our model, using the features we selected we built few models and used the Mean Square Error and the Mean Abolute Error to establish how accurate is our model in making predictions.

  • We employed two models using Descision Tree and Random Forest to make predictions, we used we used the Random Forest for as our final algorithm.

  • We used Flask API and Amazone EC2 instance to deploy the model and make it accessible to end users.

In doing this project we understood through our analysis which weather conditions affected the load shorfall, we can be able to use this to predict the shortfall and be able to employ necessary measures to close this shortfall.
