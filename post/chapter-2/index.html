Project 2: Classification of Tweets

June 30, 2023

In this project we were given a dataset that consisted of Tweets, we had to work on a classification model to classify whether a Tweet is from a user who believes in man-made climate change or not, this project is intense on the Natural Language Processing techniques. It was a great project to work on and it made me excited on the unending opportunites that Machine Learning present.

  • Began with Data Cleaning to remove all duplicates, punctuations, url links and special symbols that were not necessary for our model.

  • Did visualisation on the data, we spotted few patterns that were useful for our analysis. Spotted words that are mostly used by different categories of users.

  • Worked on tokenization and lemmatization of the tweets so that the vectorization of our tweets would be easier.

  • On this project we used a count vector to vectorize and create a bag of words, this turns the words into numerical values so that it can be understood by our algorithms

  • Used Random Forest Classifier and Support Vector Machine(SVM) Algorithms to build our model but for our problem the SVM proved to be a great solution.

  • Prepared the Classification report to gauge how our model is performing, the model proved to show a great F1 score. Which means is able to classify whether a person belives in climate change as a man-made phenomenon or not.

This has been a great project, made me realize that there are many places that sentiment analysis can be applied on, whether in trying to understand what people think about a brand or making informed decisions in the stock market by understanding the sentiments of a particular sector of the economy. I will doing more of these sentiment analysis projects in the future.
