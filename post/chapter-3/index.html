Project 3: Language Identification Program

July 1, 2023

Also enjoyed doing this project it’s a very simple project, I had dataset from NCHLT Text Corpora collected by the South African Department of Arts and Culture & Centre for Text Technology that contained 11 official South African languages. I had to build a program that will classify a sentence as to which language does it belong to.

  • As with all project we started by cleaning the data we will be using to train our model. I removed all duplicates, special characters and symbols that will skew our model.

  • As this was a classification problem I needed to inspect the class imbalance, this will help me when building the model, this helps the model that it does not become good in classifying the class that has large volume of training data.

  • I tokenized the sentences removed punctuations and stopwords and proceeded to lemmatized the sentences. I used lemmatization in this project because lemmatization retains the true meaning of the word.

  • Had to encode the target variable to use in the model, since we are dealing with natural language processing we then vectorized the sentences into numerical values that can be used in our model, the count vector seemed appropriate for this task.

  • When building the model we tried a few model to see which model would work well for us. The models we tried were Random Forest Classifier, Multinomial Naive Bayes Model and the Support Machine Vector Model. The model that seemed to give us a more accurate classification was the Support Machine Vector.

  • The score used to see if the model was classifying the data as intended we used the F1 Score, and the model gave us an F1 score of 98% on the unseen test data.

This was a great project for natural language processing and it left me wanting to creat more natural language processing projects.
