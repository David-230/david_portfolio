Projects

I will be showcasing some of the projects I have worked on, some of the project may contain sensitive data therefore I will not be sharing the data. I will outline the main points of the project and add a Git Hub repo where possible.

July 1, 2023

Project 4: Movie Recommender System

This has to be one of the intense project I have worked on in my early days of Data Science, I was in a team that was tasked to develop a movie recommender system using the unsupervised technique called recommender systems. We built this recommender system and deployed it using streamlit and made it accessible by internet using AWS. We used movie lens dataset to complete this project, we downloaded the multiple files for the movie lens dataset and used them to accomplish our project.
July 1, 2023

Project 3: Language Identification Program

Also enjoyed doing this project it’s a very simple project, I had dataset from NCHLT Text Corpora collected by the South African Department of Arts and Culture & Centre for Text Technology that contained 11 official South African languages. I had to build a program that will classify a sentence as which language does it belong to. As with all project we started by cleaning the data we will be using to train our model.
June 30, 2023

Project 2: Classification of Tweets

In this project we were given a dataset that consisted of Tweets, we had to work on a classification model to classify whether a Tweet is from a user who believes in man-made climate change or not, this project is intense on the Natural Language Processing techniques. It was a great project to work on and it made me excited on the unending opportunites that Machine Learning present. Began with Data Cleaning to remove all duplicates, punctuations, url links and special symbols that were not necessary for our model.
